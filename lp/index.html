Minstrel JPの3つの特徴

🤖

AIエージェントによる自動分析

タスクを入力するだけで、6つのAIエージェントが最適なプロンプト構造を自動設計します。

🧩

モジュール式プロンプト設計

11種類のモジュール（背景、目標、制約など）を組み合わせて構造化されたプロンプトを生成します。

🔄

反復改善システム

Commentator & Reflectorエージェントが、多角的に分析して自動改善します。

簡単3ステップで高品質なプロンプトを生成

1
📝

タスクを入力

「商品レビューを分析したい」など、やりたいことを自然な文章で入力します。

→
2
⚙️

モジュール生成

AIが自動でプロンプトの構成要素を生成。必要に応じて手動調整も可能です。

→
3
✨

プロンプト完成

統合されたプロンプトをテストして、さらに改善サイクルを回すことも可能です。

6つのAIエージェントが連携

Analyzer

→

Designer

プロンプト内容を生成

→

Simulator

プロンプトをテスト実行

→

Questioner

改善点を抽出

→

Commentator

多角的に評価
(批判×2、好意×2、中立×1)

→

Reflector

フィードバック反映

✓ 初期化（Initialization）
